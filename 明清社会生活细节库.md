**茄子鲗详细做法（第四十一回）**：
> 凤姐道："这也不难。你把才下来的茄子把皮刨了，只要净肉，切成碎钉子，用鸡油炸了，再用鸡脯子肉并香菌、新笋、蘑菇、五香腐干、各色干果子，俱切成钉子，用鸡汤煨了，将香油一收，外加糟油一拌，盛在瓷罐子里封严，要吃时拿出来，用炒的鸡爪一拌就是。"

## 二、服饰文化（服饰）

> **研究说明**：本节综合维基百科·钿子、清代宫廷服饰文化展、北京艺术博物馆服饰展、《红楼梦》服饰研究论文等权威资料整理。
> 清代贵族女性服饰呈现**满汉并行**的格局：满族女性穿旗装（长袍+旗鞋），汉族女性保留明代袄裙制度。曹雪芹时代的贾府正值康乾盛世，服饰描写融合了两种传统。

### 2.7 满族女性常服与礼服区别

| 场景 | 常服（旗装） | 礼服（吉服/朝服） |
|------|------------|----------------|
| 构成 | 长袍+旗鞋+旗髻；可加坎肩/褂 | 吉服：钿子+吉服袍；朝服：朝冠+朝袍 |
| 场合 | 日常、居家 | 重大典礼、祭祀、婚嫁 |
| 旗装特点 | 上下一体、长可掩足；领口/袖口/衣襟多花绦镶边 | 隆重正式 |
| 旗髻演变 | 清初→二把头（中期）→大拉翅（清末） | 钿子（日常）→吉服冠（正式）→晚期钿子取代吉服冠 |
| 曹雪芹时代 | 乾隆年间：两把头为贵妇日常；钿子用于吉庆场合 | — |

## 调研进度记录

### 2026-04-07 服饰文化详细研究

**来源**：Tavily 搜索（维基百科·钿子、清代宫廷服饰文化展、北京艺术博物馆服饰展、凤凰网红学文章等）

**核心发现**：

#### 1. 钿子体系（最重要发现）
- 钿子不是"旗头"，是独立冠饰系统：框架（金属丝/藤）+ 覆盖层 + 钿花
- 分三种等级：**满钿**（7块钿花，年轻/婚嫁）/**半钿**（5块，老年/寡妇）/**凤钿**（凤凰纹样）
- ⚠️**重要纠正**：乾隆时代（红楼梦时代）没有"大拉翅"旗头！大拉翅是清末（道光后）才出现的
- 曹雪芹时代贵妇日常戴的是**两把头**，钿子用于吉庆场合
- 影视中的"大旗头板"实为慈禧晚年发明，不应出现在乾隆背景的红楼梦续写中

#### 2. 霞帔规制
- 明代：窄长条带（宽3.2寸/长5尺7寸），下端坠玉石/金银
- 清代：变成背心样式，前胸后背缀**补子**（与官服补子对应品级）
- 民间婚嫁"凤冠霞帔"只是**借名**，真正霞帔是命妇礼服
- 霞帔品级：一品金绣云霞翟文 → 八九品大红素罗

#### 3. 贾府女性服饰真实还原
- **凤姐**三套服饰可作为续写模板：
  - 正装（第3回）：缕金百蝶穿花大红洋缎窄褃袄 + 五彩刻丝石青银鼠褂 + 翡翠撒花洋绉裙
  - 家常（第6回）：桃红撒花袄 + 石青刻丝灰鼠披风 + 大红洋绉银鼠皮裙 + 昭君套 + 攒珠勒子
  - 素服（第68回）：月白缎袄 + 青缎披风 + 白绫素裙（国孝在身）
- **宝钗**"半新不旧"暖色袄裙代表汉族大家闺秀日常
- **湘云**满身皮草（第49回）：貂鼠脑袋面子+水红妆缎狐肷褶子 = 满族贵族女性特征
- **袭人/鸳鸯**等丫鬟服饰：银红袄 + 青缎背心 + 白绫细折裙 = 丫鬟装束规范

#### 4. 续写防坑提醒
- ❌ 不要写"大拉翅"旗头 → 乾隆时代没有
- ❌ 凤姐的"旗头"应写"钿子"或"旗髻"
- ✅ 冬季写"昭君套""勒子"更符合原著时代
- ✅ 披风/斗篷/鹤氅是高频道具，场景广泛

### 2026-04-07 饮食文化初步研究

**来源**：Tavily 搜索 + 学术PDF

**核心发现**：
1. 红楼梦反映的是**满汉融合**的清代贵族饮食
2. 满族饮食特点：粘食、奶制品（酥酪、奶子）、肉类为主
3. 茄鲗做法是红楼梦饮食描写最详细的菜品（第四十一回）
4. 茶文化分为贵族茶席（妙玉）、文人茶席（宝黛钗）、寒门茶饮三个层次
5. "克什"是满语，指赏赐，出现在第118回
6. 满族姑娘在家地位尊崇——"小姑"习俗

**重要参考PDF**：
- 清代宫廷饮食文化的人类学解读 - PolyU（分析了肉类在上层社会的地位隐喻）
