# 红楼梦续写
## Dream of the Red Chamber — Continuation

基于脂砚斋批语，探佚曹雪芹后四十回原意。

### 创作原则

1. **伏线必应** — 前八十回所有伏笔，必须在后四十回中照应
2. **性格统一** — 人物行为必须符合其在前八十回中的性格逻辑
3. **基调一致** — "悲金悼玉"，非程高本的大团圆结局
4. **语言逼近** — 文风尽可能接近曹雪芹笔法
5. **结构完整** — 与前八十回形成对称结构

### 脂砚斋透露的迷失稿子

- 狱神庙相逢（茜雪、红玉慰宝玉）
- 花袭人有始有终
- 卫若兰射圃
- 薛宝钗借词含讽谏
- 王熙凤知命强英雄
- 悬崖撒手

### 核心伏笔来源

- 金陵十二钗判词
- 红楼梦曲
- 元春省亲四出戏
- 清虚观打蘸三出戏
