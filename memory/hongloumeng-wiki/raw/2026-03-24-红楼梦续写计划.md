# 红楼梦续写工程：阶段计划

## 项目目标
系统研究红楼梦前八十回，完成高质量后四十回续写，**还原曹雪芹原意**，符合"悲金悼玉"基调。

### 核心原则
- **严格依循曹雪芹原意**，非程高本续书
- 以脂砚斋批语为探佚核心依据
- 以伏笔为线索，还原后四十回可能面貌
- 避免程高本的问题：戏剧化冲突、人物性格扭曲
